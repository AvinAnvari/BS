# Privacy Policy

HBF is a Google Sheets™ add-on that creates a Google Docs™ document
using content provided by the user.

## Data Collection and Use
HBF does not collect, store, or transmit personal data.
The add-on only accesses data explicitly provided by the user
in the active Google Sheets™ spreadsheet (cells A1–C1).

## Data Sharing
HBF does not share user data with third parties.

## Data Storage
No user data is stored outside the user’s Google account.

## User Control
The add-on only runs when initiated by the user and only accesses
the currently active spreadsheet.
